###清分：
函数功能：对每天的账目进行清分
异常日志：保存在 工程目录/log 目录下
json对账文件：保存在 工程目录/account 目录下
返回：无
| 参数类型 | 参数名 | 参数描述 | 备注 |
| ---- | ---- | ---- | ---- |
| int | merchantID | 商户账户ID | 无 |
| float | amount | 转账金额 | 无 |
| int | operatorID | 操作ID | 无 |


###查询：
函数功能：查询数据库中的记录
返回：json文件
| 参数类型 | 参数名 | 参数描述 | 备注 |
| ---- | ---- | ---- | ---- |
| Date | from | 起始时间 | 无 |
| Date | to | 终止时间 | 无 |
| int | operatorID | 操作类型 | 无 |

###下载对账文件：
函数功能：获取在一定时间范围内的json对账文件
返回：json文件
| 参数类型 | 参数名 | 参数描述 | 备注 |
| ---- | ---- | ---- | ---- |
| Date | from | 起始时间 | 无 |
| Date | to | 终止时间 | 无 |
